What we do

SFEvents searches the web for events taking place all around San Francisco and list them on this website in an organized, interactive fashion.

Why this matters

We do the searching for you! No longer do you have to manually search for events in San Francisco; we list them all here! This saves you time and effort because you no longer need to manually search for San Francisco-specific events!

Getting started

To get started, just scroll down! You will see six up-and-coming events in San Francisco, however there will soon be many more!

The start and end time of the events, the calendar date, location, description, and ticket/purchase link are all listed within a grid.
